# Template Skill

## Objective

Define one clear workflow and execution pattern.

## Steps

1. Replace `name` and `description` in frontmatter.
2. Write procedural instructions for the workflow.
3. Add 2-3 realistic examples.
4. Document edge cases and failure handling.
5. Add optional `scripts`, `references`, or `assets` only if needed.
