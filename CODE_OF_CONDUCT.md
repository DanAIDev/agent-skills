# Code of Conduct

## Our Commitment

This project is committed to a respectful, harassment-free environment for everyone.

## Expected Behavior

- Be respectful and constructive.
- Focus on technical issues, not people.
- Accept and provide feedback professionally.

## Unacceptable Behavior

- Harassment, personal attacks, or discrimination.
- Trolling, intimidation, or deliberate disruption.
- Sharing private information without permission.

## Enforcement

Project maintainers may remove comments, issues, or contributions that violate this policy.
